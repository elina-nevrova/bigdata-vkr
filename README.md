**Облачная платформа для владельцев 3D-принтеров и малых производственных ферм, продающих изделия на маркетплейсах.**

Сервис объединяет в одном интерфейсе три инструмента: рыночную аналитику аддитивного производства на маркетплейсах, калькулятор полной себестоимости 3D-печати и ML-рекомендации по выбору прибыльных ниш.


## Проблема

Владельцы 3D-ферм и частные мастера, продающие изделия на маркетплейсах:

- тратят до **20 часов в неделю** на ручной анализ ниш;
- считают цены «на глаз», не учитывая себестоимость филамента, время печати, брак (5–10 %), амортизацию, актуальные комиссии площадок и логистику;
- теряют **30–50 % выручки** из-за затоваривания невостребованными моделями и демпинга;
- тратят десятки тысяч рублей в месяц на тестовые тиражи, которые не окупаются.

Существующие сервисы аналитики ориентированы на перекупщиков и не учитывают специфику аддитивного производства, а калькуляторы печати не связаны с рыночными данными.

---

## Решение

Пользователь задаёт параметры своего производства (принтеры, материалы, время печати, брак, электроэнергия, упаковка) — сервис:

1. **Рассчитывает полную себестоимость** изделия, минимальную цену продажи и маржу с учётом комиссий, логистики, хранения и продвижения.
2. **Анализирует рынок:** цены, количество карточек, отзывы, динамику цен и индекс конкуренции по товарным категориям.
3. **Рекомендует ниши:** ранжирует ≥ 20 товарных изделий по прогнозируемой доходности именно для фермы конкретного пользователя.
4. **Позволяет проверять сценарии «что если»:** изменение цены материала, времени печати, процента брака, цены продажи и т. д. — до запуска тиража.

## Модули

| Модуль | Возможности |
|---|---|
| **Калькулятор себестоимости** | Профили принтеров (≥ 3) и материалов (≥ 10); полная себестоимость, минимальная цена, валовая прибыль и маржинальность; расчёт сценария ≤ 10 сек |
| **Мониторинг рынка** | Wildberries, ≥ 10 категорий, ≥ 10 000 товарных карточек и 100 000 записей рыночных показателей в первой версии |
| **Рекомендатор (ML)** | Персонализированный топ-20 ниш: себестоимость, ценовой диапазон, конкуренция, прогнозная маржа, объём тестовой партии |
| **Отчёты и сценарии** | Хранение расчётов, сравнение сценариев цены/объёма, экспорт в табличном формате |

## Технический стек

Сервис построен из четырёх слоёв.

### 1. Слой сбора данных (Data Collection Layer)
- **Python (Scrapy, Playwright, patchright)** — сбор данных Wildberries: SKU, категории, продажи, рейтинги, цены, динамика.
- **PostgreSQL** — хранение оперативных и транзакционных данных.
- **ClickHouse** — колоночная СУБД для аналитических запросов.
- Ежедневное обновление по расписанию.

### 2. Слой обработки и аналитики (Analytics Layer)
- **Apache Airflow** — ETL-пайплайн: нормализация, расчёт метрик (оборачиваемость, доля рынка, индекс конкуренции, сезонность).
- **FastAPI (Python)** — REST API бэкенда.

### 3. ML-слой
- **Прогнозирование спроса:** временные ряды — Prophet + LightGBM / CatBoost / PyTorch.
- **Рекомендательная система:** коллаборативная фильтрация с учётом производственного профиля пользователя.
- Переобучение моделей — еженедельно на накопленных данных.

### 4. Фронтенд (User Interface)
- **Vue.js** + интерактивные дашборды.

## Инфраструктура разработки

- **Docker** — контейнеризация сервисов.
- **GitHub** — приватный репозиторий (300+ коммитов прототипа).
- **VS Code**, **Figma** — разработка и дизайн.
- Тестовый сервер: **4 vCPU / 8 ГБ**.
- Два принтера **Bambu Lab P2S** с AMS для верификации калькулятора (протестировано 30 моделей PLA/ABS).
